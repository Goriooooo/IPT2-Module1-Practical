#Eris Cafe – Full-Stack Cafe Management System
A modern, full-stack cafe management web application built with the MERN stack (MongoDB, Express, React, Node.js) + TypeScript. Features online ordering, table reservations, user authentication, and a powerful admin dashboard.
Features

Customer Features
User registration & login (JWT-based auth)
Browse menu & place orders
Book table reservations
View order & reservation history

Admin Dashboard
Real-time statistics (total users, orders, reservations, revenue)
Manage orders (pending → processing → completed)
Manage reservations (confirm/reject)
View analytics and reports

Tech Stack
Frontend: React + TypeScript + Tailwind CSS (or your chosen UI library)
Backend: Node.js + Express + TypeScript
Database: MongoDB with Mongoose
Authentication: JWT (access tokens expire in 1 hour)
Role-based access: customer and admin
